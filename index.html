Google Drive Setup Guide

One-time setup to enable Drive sync:

  1. Go to console.cloud.google.com → New Project
  2. Enable Google Drive API in the API Library
  3. Go to Credentials → Create OAuth 2.0 Client ID → Web Application
  4. Add Authorized JavaScript Origins:
    http://localhost:8080 and http://127.0.0.1:8080
  5. Copy the Client ID and paste it above
  6. Serve Ferdowsi locally: python3 -m http.server 8080
  7. Open http://localhost:8080/Ferdowsi.html
